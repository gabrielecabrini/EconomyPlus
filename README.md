## Maven Instructions [![](https://jitpack.io/v/ItsWagPvP/EconomyPlus.svg)](https://jitpack.io/#ItsWagPvP/EconomyPlus)

### Add Repository
	<repositories>
	    <repository>
	        <id>jitpack.io</id>
	        <url>https://jitpack.io</url>
	    </repository>
	</repositories>
 
### Add Dependency
	<dependency>
	    <groupId>com.github.ItsWagPvP</groupId>
	    <artifactId>EconomyPlus</artifactId>
	    <version>4.9</version>
	</dependency>
